# UgraNeft VR
VR-игра для изучения нефтяной промышленности.
Пользователю предстоит проходить интерактивные сценарии, знакомиться с ключевыми процессами нефтедобычи, работы оборудования и правилами безопасности на производстве.

UgraNeft VR - это обучающая VR-игра, в которой вы погружаетесь в мир нефтяной промышленности и осваиваете профессиональные навыки в виртуальной среде при помощи виртуального помощника Романа Ивановича Кузоваткина. Вас ждут реалистичные сценарии, связанные с добычей и переработкой нефти, а также с эксплуатацией оборудования и соблюдением техники безопасности. Приложение ориентировано на практическое обучение и помогает лучше понять рабочие процессы отрасли в наглядном и интерактивном формате.

В релизах находятся 2 модуля которые можно скачать  в виде .exe файлов. Для управления персонажем нужно VR оборудование!

Технология и языки программирования:
- Varwin - платформа основанная на Unity engine
- Язык для логики/скриптов и SDK — C#
- Некоторые 3d модели и ассеты сделанны в Blender

Будущие релизы:
- Запуск буровой вышки (модуль)
- Запуск насоса-качалки (модуль)
- Осмотр месторождения (модуль)
- Лаборатория (модуль)
